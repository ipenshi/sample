# sample
githubの利用、react-nativeの実装、稼動、etc…　練習用のリポジトリ。練習用。

## 初期設定  
### ◆環境  
  - node: 10.9.0  
  - watchman: 4.9.0  
  - react-native-cli: 2.0.1  
  - react-native: 0.57.0  

### ◆開発リポジトリ取得方法
cdでDesktopに移動した上で以下のコマンドを実行  
  1. git clone https://github.com/ipenshi/sample.git  

以上！
  
### ◆修正ファイルコミット方法
以下のコマンドを実行  
  1. git commit -m "コメント"  
  1. git push  
  
以上！  
①でローカルリポジトリにコミット、②でＷEB上のリポジトリにコミットしている。
